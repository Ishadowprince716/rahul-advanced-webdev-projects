# AI-Powered Dashboard

This project is a full-stack web application that displays real-time analytics using an AI backend.

## Features
- User authentication and authorization
- Interactive dashboard with charts
- AI-powered forecasts
- Cloud database integration

## Stack
- React
- Node.js
- MongoDB
- Streamlit for AI module
- JWT security

## Getting Started
1. Clone the repo
2. `npm install` and run React/Node servers
3. Start Streamlit backend for AI
