# Blockchain Voting Platform

This is a decentralized voting application utilizing Ethereum smart contracts for secure, transparent elections.

## Features
- Voter registration/authentication
- Voting logic using Solidity smart contracts
- Results dashboard
- Web3.js frontend

## Stack
- React
- Node.js
- Solidity
- Web3.js
- MetaMask Integration

## Getting Started
1. Clone repository
2. Setup Node/React frontend
3. Deploy contracts using Truffle/Hardhat
