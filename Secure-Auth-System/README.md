# Cybersecurity Auth System

Production-ready authentication and user management system with robust security practices.

## Features
- Multi-factor authentication (MFA)
- Secure password storage (bcrypt)
- Role-based access control
- Logging and alerting

## Stack
- Django
- PostgreSQL
- JWT/OAuth2

## Getting Started
1. Clone the repo
2. Setup Django backend with PostgreSQL
3. Use `/auth` endpoints for user management
